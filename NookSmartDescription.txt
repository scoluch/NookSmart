Installation and Setup:
1.	Unplug the light connector (two wires) from the battery pack and plug it into the small socket on the NookSmart.
2.	Unplug the touch switch connector (three wires) from the battery pack and plug it into the large socket on the NookSmart. If your book nook does not have a touch switch, you can leave this socket unplugged.
3.	Connect the USB-C cable to the NookSmart and connect it to power.

WiFi Setup:
1.	The LED lights will immediately start blinking quickly. 
2.	Press the setup button with a paperclip two times within 5 seconds.
3.	The lights should start to blink slowly.
4.	When the lights are blinking slowly, connect your phone or other device to the WiFi network called NookSmart-Setup. If asked for a WiFi password, enter 12345678.
5.	Open a web browser and enter 192.168.4.1
6.	You should see the setup screen. Select your WiFi network and enter its password. Below that set the name for the device and set the gamma value to scale brightness then click to save your settings. See below for on-device scheduling.
7.	Press the setup button again once or unplug and re-plug the USB to restart.

Apple Home Setup:
1.	Open the Home app and choose Add Accessory.
2.	Scan this QR code:
 
3.	Follow the on-screen prompts to add the device to Apple Home – it should use the name you set in the previous step.
4.	If the QR code does not work, you can manually select and add the device using pairing code 4663-7726.
5.	Once setup, you can say things like “Siri, turn on [name]”, “Siri, turn off [name]”, and “Siri, set [name] to 50%”

Amazon Alexa Setup:
1.	Say “Alexa, search for new devices” 
2.	After a few moments it should locate your device and name it the same name you set in the main setup.
3.	Once setup, you can say things like “Alexa, turn on [name]”, “Alexa, turn off [name]”, and “Alexa, set [name] to 50%”

On-Device Scheduling
You can set your own schedules for on/off and brightness right in the setup portal. 
1.	Set your time zone for accurate time – the device will automatically keep time when this is set.
2.	Click or tap the Add Schedule button. 
3.	Set the time and days of the week, then choose the action. If you select Turn On, you can also set the brightness. 
4.	Click or tap the trash can icon to delete a schedule event
5.	Toggle the Enabled option to temporarily turn off an event without deleting it
6.	Add more events by clicking the Add Schedule button. You can add up to 10 events.
7.	Click or tap Save Settings at the bottom and let the device restart for the schedule to take effect.
